## 🚀 Flagship Project — SESKO AI

> India's first privacy-first, edge-native Sovereign AI Agent

**The problem:** Most AI assistants require sending your data to someone else's cloud. For Indian users, SMBs, and anyone handling sensitive data, that's a non-starter.

**The bet:** Build an assistant that runs entirely on local hardware — zero cloud dependency — without sacrificing capability.

```
SESKO AI Stack
├── 🧠 LLM Engine      → Llama 3.2 (Local)
├── ⚡ Backend         → FastAPI (Python)
├── 📊 Analytics       → SQLite + Predictive ML Engine
├── 👁️ OCR Module      → Document Intelligence
├── 💹 Trading Engine  → SESKO-ON.AI Predictive Module
└── 🔒 Privacy         → No Cloud. No Compromise.
```

**What I'm tracking next:** latency vs. cloud-LLM benchmarks, and which user segment (solo professionals vs. small businesses) gets the most value first — the classic AI PM question of *who do we build for next*.
