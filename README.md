## 🚀 What I'm Building

### 🤖 SESKO AI — Sovereign AI Agent
> India's first privacy-first, edge-native AI assistant built for **data sovereignty**

- Built with **Python + FastAPI + Llama 3.2**
- Runs 100% on **local hardware — zero cloud dependency**
- Features: OCR, B2B Intelligence Engine, Predictive ML Engine
- Trading & analytics module powered by **SQLite**
- Architecture: **Hybrid Sovereign Architecture**

```
SESKO AI Stack
├── 🧠 LLM Engine      → Llama 3.2 (Local)
├── ⚡ Backend         → FastAPI (Python)
├── 📊 Analytics       → SQLite + Predictive ML
├── 👁️ OCR Module      → Document Intelligence
├── 🔒 Privacy         → No Cloud. No Compromise.
└── 🇮🇳 Built for India → Data Sovereignty First
```
